## Списки

Чтобы добавить ненумерованые списки, необходимо выделить пункт списка звездочкой (*).
Например вот так:

* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованые списки, необходимо пункты списка пронумеровать (1, 2, 3...). Например вот так:

1. Первый пункт
2. Второй пункт
